About

Bruh, Ltd is a collection of random web pages, such as demos and art projects, that were created for questionable reasons. All of the pages are placed in a sequencial order and linked to each other with secret links, which you should navigate. This page contains one too.

The bruh.ltd domain also hosts instances and landing pages of some projects. Feel free to guess how you can find them out.

If you have a random web page and you believe it belongs here, feel free to submit a merge request. Your page will gladly be accepted as long as it's under a compatible license.
